# TikTok Label Formatter - Technical Prototype

## Overview
A client-side utility to transform TikTok Shop's A4 shipping label PDFs into 4x6 thermal-ready formats.

## Structure
- `/src/` - Processing logic (PDF-Lib / WASM)
- `/public/` - Static landing page
- `/test/` - Sample "broken" labels for validation
